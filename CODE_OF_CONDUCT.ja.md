# crop 行動規範

言語: [English](CODE_OF_CONDUCT.md) | [한국어](CODE_OF_CONDUCT.ko.md) | [简体中文](CODE_OF_CONDUCT.zh-CN.md) | 日本語

この文書は日本語訳です。言語間で意味が異なる場合は English 版を正とします。

`crop` リポジトリは、小さく実用的で、互いを尊重するオープンソース協力の場を目指します。この行動規範は、Issues、Pull Requests、Discussions、code review、ドキュメント、リリース準備の会話など、リポジトリに関連するすべての公開のやり取りに適用されます。

## 期待される行動

- 他者を尊重し、善意を前提にします。
- 主張には再現手順、根拠、tradeoff を添えます。
- 批判は人ではなく、コード、文書、方針、ユーザー影響に集中します。
- 他者の時間と文脈を尊重し、依頼を明確にします。
- security、privacy、license、branding の話題は慎重に扱い、検証可能な根拠を優先します。
- 誤りを見つけた場合は、修正への道筋を提案します。

## 許容されない行動

- 侮辱、嫌がらせ、脅迫、威圧
- 差別的な表現や排他的な冗談
- 不適切な性的または暴力的な内容
- 同意なく個人情報、非公開の連絡先、機密性の高いセキュリティ情報を公開する行為
- 繰り返される off-topic な議論、spam、意図的な妨害
- maintainer や contributor に対して不合理な応答時間、判断、実装を強要する行為

## 報告と対応

行動規範違反を見つけた場合は maintainer に知らせてください。

- 公開で扱える内容は、GitHub Issue または Discussion に簡潔に報告できます。
- 個人情報、安全上の懸念、機密性の高いセキュリティ詳細を含む場合は、詳細を公開しないでください。
- 非公開での調整が必要な場合は、GitHub profile に表示されている連絡経路で maintainer `@postmelee` に連絡するか、「非公開フォローアップが必要な行動規範報告」のような最小限の公開メモだけを残してください。

Maintainer は必要に応じて次の対応を取ることがあります。

- 会話を本題に戻すよう依頼する
- コメントの編集または非表示を依頼する
- Issue、Pull Request、Discussion をロックする
- 繰り返し違反する人の参加を制限する
- GitHub プラットフォームの報告ツールで追加対応する

可能な範囲で、情報は必要な人にのみ共有します。公開済みのコメントや活動記録は、GitHub のプラットフォームポリシーと権限モデルの範囲内でのみ調整できます。

## 適用範囲と限界

この文書は協力の最低限の期待事項を定めます。法的助言、雇用ポリシー、プラットフォーム規則の代替ではありません。GitHub Terms、GitHub Community Guidelines、リポジトリのライセンス、セキュリティポリシーも適用される場合があります。
